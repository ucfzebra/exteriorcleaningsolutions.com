# Exterior Cleaning Solutions

Website for **Exterior Cleaning Solutions** — a pressure washing and exterior cleaning service.

## Overview

Single-page marketing site built with plain HTML and CSS. No build tools or dependencies required.

## Structure

```
index.html   # Main site (all HTML, CSS, and JS in one file)
LICENSE
```

## Development

Open `index.html` directly in a browser — no server or build step needed.

## Deployment

The site can be hosted on any static file host (Cloudflare Pages, Netlify, GitHub Pages, etc.) by pointing the root to `index.html`.
